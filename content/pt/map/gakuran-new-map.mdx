<Section icon={Shield} title="Controles e Teclas de Atalho para Exploração do Novo Mapa de Gakuran">

Para navegar pela complexa verticalidade e pelos cantos apertados do novo mapa, você precisa dominar perfeitamente o seu esquema de controle. Gakuran utiliza os controles padrão do Roblox juntamente com comandos personalizados de briga. Não importa se você joga no PC, celular ou usando um controle, dominar essas teclas garantirá que você possa se esquivar de golpes recebidos e atravessar o mapa com fluidez.

A tabela abaixo descreve o layout de controle padrão em todas as plataformas compatíveis. Certifique-se de praticar esses comandos em uma zona de baixo risco antes de entrar em combate ativo.

| Ação | Teclado e Mouse | Gestos no Celular | Entrada do Controle | Nota Tática de Combate |
| :--- | :--- | :--- | :--- | :--- |
| **Movimentação** | WASD / Setas | Analógico na Tela | Analógico Esquerdo | Essencial para espaçamento e corte de ângulos |
| **Pular** | Espaço | Botão de Pular na Tela | Botão A | Use com moderação no combate para evitar golpes perdidos |
| **Correr / Arrancada** | Toque duplo no W | Alternador de Corrida na Tela | L3 / Clique no Analógico Esquerdo | Crucial para escapar de zonas de alto risco |
| **Esquivar / Dash** | Q | Botão de Dash na Tela | Botão B | Perfeito para reposicionamento durante os combos |
| **Bloquear / Parry** | F | Botão de Bloqueio na Tela | Gatilho L2 | Segure para bloquear; acerte o tempo perfeitamente para dar parry |
| **Postura de Combate** | R | Botão de Alternar Combate | Direcional Direito | Deve estar ativa para executar golpes de briga |
| **Ataque Leve** | M1 / Clique Esquerdo | Toque na Tela | Gatilho R2 | Iniciador rápido para sequências de socos e chutes |
| **Ataque Pesado** | M2 / Clique Direito | Segurar Botão de Ataque | Botão Y | Ferramenta de quebra de guarda com alto tempo de recuperação |
| **Carregar Oponente** | B | Carregar na Tela | Botão X | Usado para transportar inimigos derrotados |
| **Executar / Agarrar** | V | Executar na Tela | Botão L1 + X | Finaliza os confrontos de combate |

<Callout type="success" title="Otimização do Shift Lock">
Sempre ative a opção "Shift Lock Switch" no menu de configurações do seu Roblox. Pressionar Shift no teclado permite que você trave a câmera no movimento do seu personagem, tornando significativamente mais fácil circular o adversário e acertar esquivas laterais no PvP.
</Callout>
