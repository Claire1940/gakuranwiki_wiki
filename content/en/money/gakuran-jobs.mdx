<Section icon={Shield} title="Mastering the Controls and Keybinds">

To excel at any of the combat-related **gakuran jobs**, you must have muscle memory for the keybinds. The game utilizes the standard Roblox engine but adds custom layers for the fighting mechanics. Whether you are on PC, Mobile, or Controller, knowing your inputs is non-negotiable.

| Action | PC Keybind | Mobile Input | Controller |
| :--- | :--- | :--- | :--- |
| **Move** | WASD | Virtual Joystick | Left Stick |
| **Combat Stance** | **R** | On-screen Toggle | D-Pad Up |
| **Light Attack** | M1 (Left Click) | Tap Screen | R2 / RT |
| **Heavy Attack** | M2 (Right Click) | Hold / Button | Y / Triangle |
| **Block / Parry** | **F** | Shield Icon | L2 / LT |
| **Dodge / Dash** | **Q** | Dash Button | B / Circle |
| **Execute / Grab** | V | On-screen Button | L1 + X |
| **Carry Player** | B | On-screen Button | X / Square |

<Callout type="tip" title="Stamina Management">
Dashing (Q) and Heavy Attacks (M2) consume stamina. If you deplete your bar, you will be unable to block, leaving you completely vulnerable. In Gakuran, the player who manages their stamina better usually wins the "job" of the fight.
</Callout>
