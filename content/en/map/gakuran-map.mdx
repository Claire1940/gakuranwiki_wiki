<Section icon={Swords} title="Core Controls & Combat Mechanics">

Combat in Gakuran is fast-paced and relies heavily on spacing, timing, and defensive discipline. Because the game runs on the core Roblox engine, mastering the default control layer is your first step toward surviving open-world encounters. Entering your combat stance is mandatory before you can execute offensive strings or defend yourself from incoming strikes.

The table below outlines the default controls for PC, mobile, and controller players.

| Action | Keyboard & Mouse | Mobile Gestures | Controller Input |
| :--- | :--- | :--- | :--- |
| Move | WASD / Arrow keys | On-screen joystick | Left thumbstick |
| Jump | Spacebar | On-screen jump button | A button |
| Camera | Right-click and drag | Swipe on screen | Right thumbstick |
| Combat Stance | R key | In-game menu option | Help menu shortcut |
| Light Attack | Left-click (M1) | Tap screen | Right trigger (R2) |
| Heavy Attack | Right-click (M2) | Tap screen | Y button |
| Block / Parry | F key | On-screen block icon | Left trigger (L2) |
| Dodge / Dash | Q key | On-screen dash icon | B button |
| Carry Opponent | B key | Touch prompt | X button |
| Execute / Grab | V key | Touch prompt | Left bumper + X |

To win fights consistently, you must combine these basic inputs into structured patterns. Avoid swinging wildly; instead, wait for your opponent to miss a heavy attack, block their incoming strike, and then unleash a quick light-attack combo.

<Callout type="warning" title="Combat Warning">
Do not hold your block button indefinitely. Experienced players can easily break your guard with a heavy attack, leaving you stunned and completely vulnerable to high-damage combos.
</Callout>
