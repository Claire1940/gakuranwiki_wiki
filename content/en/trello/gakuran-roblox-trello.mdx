<Section icon={Shield} title="Combat Controls and Keybinds Mastery">

To excel in Gakuran, you must move beyond basic movement. The game utilizes the standard Roblox engine but implements custom scripts for combat stances, dashing, and executions. Mastering the "Shift Lock" feature is mandatory, as it allows for the directional accuracy required to land heavy attacks and parries.

The following table breaks down the essential keybinds for PC, Mobile, and Controller players. Note that the "Combat Stance" (R) is the most important button; without it, your character remains in a "passive" state and cannot effectively block incoming damage.

| Action | PC Keybind | Mobile Gesture | Controller |
| :--- | :--- | :--- | :--- |
| **Movement** | WASD / Arrows | Joystick | Left Stick |
| **Combat Stance** | **R** | On-screen Toggle | D-Pad Up |
| **Light Attack** | M1 (Left Click) | Tap Screen | R2 / RT |
| **Heavy Attack** | M2 (Right Click) | Hold Screen | Y / Triangle |
| **Block / Parry** | **F** | Block Button | L2 / LT |
| **Dash / Dodge** | **Q** | Swipe | B / Circle |
| **Execute / Grab** | **V** | Context Button | L1 + X |
| **Carry** | **B** | Context Button | X / Square |

<Callout type="warning" title="Stamina Management">
Every dash and heavy attack consumes stamina. If your stamina bar hits zero during a fight, you will be "broken," leaving you defenseless against a full combo or an execution. Always watch your bar before committing to a dash-in.
</Callout>

</Section>
