<Section icon={Target} title="操作方法とキーバインドのマスター">

学乱 Gakuran Roblox RP での成功はマッスルメモリー（身体で覚えること）から始まります。このゲームは Roblox エンジン内で動作するため、標準的な移動キーを使用しますが、標準的なロールプレイゲームとは異なる複雑な戦闘入力が層として加わっています。初心者が苦戦するのは、専用の戦闘システムを使用せずに「パッシブ」モードで戦おうとするためです。

野心的な喧嘩師にとって最も重要なボタンは「R」キーで、これは戦闘スタンスを切り替えます。このスタンスに入っていないと、ブロック、ダッシュ、コンボの実行能力が大幅に制限されます。以下は、PC、モバイル、コントローラーユーザー向けの包括的な操作体系です。

| アクション | PC キーバインド | モバイルジェスチャー | コントローラー |
| :--- | :--- | :--- | :--- |
| **戦闘スタンス** | R | スタンスボタン | 方向キー上 |
| **弱攻撃** | M1 / 左クリック | 画面をタップ | R2 / RT |
| **強攻撃** | M2 / 右クリック | 長押しタップ | Y / △ |
| **ブロック / パリィ** | F | ブロックボタン | L2 / LT |
| **回避 / ダッシュ** | Q | ダッシュアイコン | B / 〇 |
| **スプリント** | W を2回押し | ジョイスティック延長 | L3 / スティック押し込み |
| **処刑 / 掴み** | V | コンテキストボタン | L1 + X |
| **プレイヤーを運ぶ** | B | 運ぶアイコン | X / □ |

<Callout type="tip" title="プロのヒント: シフトロック">
Roblox の設定で常に「シフトロック（Shift Lock）」を有効にしてください。Gakuran では、カメラをキャラクターの向きに固定することで、ムエタイやボクシングのようなテンポの速い攻防において、より正確な当たり判定（ヒットボックス）の管理が可能になります。
</Callout>

</Section>

<Section icon={Shield} title="ステップバイステップ戦闘戦略">

計画なしに学乱 Gakuran Roblox RP の戦闘に飛び込むことは、「ダウン」状態への最短ルートです。戦闘システムは、単にクリックを連打するプレイヤーよりも、相手のアニメーションを読めるプレイヤーに報います。効果的な戦闘は、誘い、パリィ、そして反撃のサイクルです。

オープンワールドの学園乱闘で勝率を上げるために、以下のステップに従ってください。

<Steps>
  <Step num="1" title="戦闘スタンスを起動する">
    **R** を押さずに敵に近づいてはいけません。これにより、即座にブロック（F）やダッシュ（Q）ができるようになります。パッシブモードで戦闘に入ると、相手の「無料の」強攻撃オープナーの標的になります。
  </Step>
  <Step num="2" title="間合いの管理 (スペーシング)">
    ダッシュ（Q）を使用して、相手の M1 攻撃の射程のすぐ外側に留まります。目標は相手の攻撃を「空振り（ミス）」させることです。強攻撃（M2）を外すと、相手は約1秒間無防備になります。
  </Step>
  <Step num="3" title="ジャストガードをマスターする">
    攻撃が当たる直前に **F** を押すとパリィが発動します。これにより相手がスタンし、相手がすぐにダッシュで逃げる隙を与えずに自分のコンボを開始できます。
  </Step>
  <Step num="4" title="処刑と再配置">
    相手をダウンさせたら、**V** で処刑するか **B** で安全な場所まで運びます。ノックダウン後は常に周囲を確認してください。近くで「漁夫の利」を狙うプレイヤーが乱入の機会を伺っていることが多いからです。
  </Step>
</Steps>

<Callout type="tip" title="スタミナ管理">
スタミナバーに注意してください！ダッシュや強攻撃は大きなエネルギーを消費します。スタミナが切れると効果的にブロックできなくなり、ムエタイのラッシュに対して完全に無防備になります。
</Callout>

</Section>
