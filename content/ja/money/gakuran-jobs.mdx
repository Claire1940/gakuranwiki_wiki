校舎は体験の中心地です。ここにはロールプレイ用の教室があり、廊下は野心的な乱闘者たちのチョークポイント（難所）になることがよくあります。街の通りはより開放的ですが、戦いを探している組織化されたクルーが頻繁にパトロールしています。

<Callout type="warning" title="危険地帯">
**廊下と入り口**は「高リスクゾーン」に分類されます。構造が狭いため、複数のプレイヤーに追い詰められやすいです。これらのエリアを移動するときは、常に脱出戦略を立てておきましょう。
</Callout>

<Section icon={Target} title="学乱の旅を始める方法">

学乱で最初の「ジョブ」を始めるには、ベテランコミュニティに圧倒されないための具体的なステップがいくつか必要です。学校内での地位を確立するために、以下の進行パスに従ってください。

<Steps>
  <Step num="1" title="公式コミュニティに参加する">
    ゲームを起動する前に、**(学乱) Gakuran Roblox Group** に参加してください。これにより、コミュニティのアップデートが提供され、コードや Trello の更新が投稿される公式ソーシャルリンクに繋がることができます。
  </Step>
  <Step num="2" title="操作設定を行う">
    ゲームに入ったらすぐに **Roblox 設定** を開きます。**シフトロックスイッチ (Shift Lock Switch)** が ON になっていることを確認してください。学乱の戦闘は、シフトロックが提供する三人称視点の精度なしにはマスターすることはほぼ不可能です。
  </Step>
  <Step num="3" title="初期スタイルを選択する">
    スタイルトレーナーまたはメニューを訪れ、最初の格闘スタイルを選択します。90% のプレイヤーにとって、**ボクシング**から始めるのが推奨される道です。複雑なギミックなしで M1/M2/F システムの核となるメカニクスを学べます。
  </Step>
  <Step num="4" title="ジムで練習する">
    ジムまたはマップの交通量の少ないエリアに向かいます。**戦闘スタンス (R)** とダッシュのタイミングを練習してください。街の通りに向かう前に、ダッシュで動きをキャンセルし、本能的にブロックできるようになっている必要があります。
  </Step>
</Steps>

<Callout type="success" title="プロのヒント：R キー">
戦闘が予想されるときは、常に**戦闘スタンス (R)** を維持してください。「カジュアル」モードで歩き回るとブロックの反応が遅くなり、不用心なプレイヤーを狙う「ランナー」の格好の標的になります。
</Callout>

</Section>

<Section icon={Shield} title="操作とキーバインドのマスター">

戦闘関連の**学乱ジョブ**で秀でるためには、キーバインドを筋肉に覚え込ませる必要があります。ゲームは標準的な Roblox エンジンを使用していますが、格闘メカニクス用にカスタムレイヤーを追加しています。PC、モバイル、コントローラーのいずれであっても、入力を知っておくことは必須です。

| アクション | PC キーバインド | モバイル入力 | コントローラー |
| :--- | :--- | :--- | :--- |
| **移動** | WASD | 仮想ジョイスティック | 左スティック |
| **戦闘スタンス** | **R** | 画面上の切り替え | 方向キー上 |
| **弱攻撃** | M1 (左クリック) | 画面タップ | R2 / RT |
| **強攻撃** | M2 (右クリック) | 長押し / ボタン | Y / 🔺 |
| **ブロック / パリィ** | **F** | シールドアイコン | L2 / LT |
| **回避 / ダッシュ** | **Q** | ダッシュボタン | B / 🔴 |
| **処刑 / 掴み** | V | 画面上のボタン | L1 + X |
| **プレイヤーを運ぶ** | B | 画面上のボタン | X / 🔵 |

<Callout type="tip" title="スタミナ管理">
ダッシュ (Q) と強攻撃 (M2) はスタミナを消費します。ゲージを使い果たすとブロックができなくなり、完全に無防備になります。学乱では、スタミナ管理が上手いプレイヤーが通常、戦闘という「ジョブ」を制します。
</Callout>

</Section>
