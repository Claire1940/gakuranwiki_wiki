<Section icon={Swords} title="学乱スタイルガイド：格闘スタイル ティアリスト">

どの戦闘規律を選択するかは、最も重要な決断です。この**学乱スタイルガイド**のセクションでは、プレッシャー性能、使いやすさ、機動力、そして全体的なPvPの有効性に基づいて、現在の格闘スタイルを分類します。現在、ゲームにはムエタイ、ボクシング、そしてメタ外の「秤（ハカリ）」スタイルの3つが登場しています。それぞれの規律は特定のプレイスタイルに対応しており、選択によって間合いの取り方、ブロックの規律、攻撃のプレッシャーのかけ方が決まります。

派手で予測不能なメタ外の選択を好むプレイヤーもいますが、学乱での競争に勝つためには、信頼できるフレームデータと強力な前進力を持つスタイルが有利です。以下の表でスタイルを比較し、コンボ練習に時間を費やす前に最適なルートを選んでください。

| スタイル | ティア | 役割 | 主な強み | 練習の重点 |
| :--- | :--- | :--- | :--- | :--- |
| **ムエタイ** | **S-Tier** | 攻撃的ストライカー | 蹴り主体の近接、素早い反撃 | ダッシュ攻撃、ブロックのタイミング |
| **ボクシング** | **A-Tier** | 堅実なブローラー | 綺麗なパンチコンボ、高い防御力 | 弱攻撃、安全な離脱 |
| **秤 (ハカリ)** | **B-Tier** | 特殊スペシャリスト | 高い奇襲性、派手な技 | ブロックの誘い、対面知識 |

<Callout type="tip" title="スタイルの再選択">
攻撃的なプレイヤーに苦戦している場合は、ボクシングへの切り替えを検討してください。その分かりやすい防御オプションと読みやすいパンチの連撃は、核心となるスペーシング（間合い管理）のメカニクスを学ぶための最高のツールです。
</Callout>

### ムエタイ (S-Tier)
ムエタイは、近接プレッシャーにおいて疑いようのない王者です。強力な蹴りベースの近接攻撃を駆使するこのスタイルは、相手のガードを崩し、迅速な反撃を繰り出すことに長けています。乱闘が頻発する混雑した廊下での戦闘において非常に効果的です。ただし、正確な間合い管理が必要であり、無謀な深追いは素早いカウンター攻撃を招く隙を作ることになります。

### ボクシング (A-Tier)
ボクシングは、学乱の中で最も初心者向けのスタイルです。非常に信頼性の高いパンチコンボと優れたブロック復帰時間を備えています。ボクシングを使用するプレイヤーは、弱パンチの連撃を容易に実行し、スムーズに防御的な離脱へと移行できます。ムエタイのような圧倒的なリーチはありませんが、その防御的な安定性はソロプレイヤーにとって非常に安全な選択肢となります。

### 秤 / ハカリ (B-Tier)
秤は、標準的なプレッシャーのかけ方よりもクリエイティブな表現を重視するプレイヤー向けに設計された、派手でメタ外のスタイルです。ユニークな攻撃角度と高い視覚的演出が特徴で、その独特なタイミングに慣れていない相手を翻弄するのに適しています。しかし、深いゲーム知識が必要とされるため、初心者にはお勧めしません。

</Section>

<Section icon={Shield} title="標準的な操作方法とキーバインド">

学乱内で効率的に動くためには、操作レイヤーを完全に理解する必要があります。このゲームはRobloxエンジン上で動作しているため、プラットフォーム全般のナビゲーションとカスタム戦闘入力の両方をマスターすることが生存に不可欠です。以下に、PC、モバイル、およびコントローラーの主要な入力をまとめました。

| アクション | キーボード ＆ マウス | モバイル・ジェスチャー | コントローラー入力 |
| :--- | :--- | :--- | :--- |
| **移動** | WASD / 矢印キー | 画面上ジョイスティック | 左アナログスティック |
| **ジャンプ** | スペースキー | 画面上ジャンプアイコン | Aボタン / × |
| **戦闘スタンス** | Rキー | 画面上切り替えボタン | 方向キー下 |
| **弱攻撃** | 左クリック (M1) | 画面ターゲットをタップ | R2トリガー |
| **強攻撃** | 右クリック (M2) | 画面上強攻撃ボタン | Yボタン / △ |
| **ブロック / パリィ** | Fキー | 画面上シールドアイコン | L2トリガー |
| **回避 / ダッシュ** | Qキー | スワイプインジケーター | Bボタン / 〇 |
| **ダウン状態を運ぶ** | Bキー | 画面上インタラクト | Xボタン / □ |
| **とどめ / 掴み** | Vキー | 画面上スカルアイコン | L1バンパー + X |

<Callout type="info" title="シフトロックの最適化">
Robloxの設定メニューで必ず「シフトロックスイッチ」を有効にしてください。PCでシフトロックを使用すると、カメラの視点を固定できるため、ダッシュの狙いを定めたり、ブロックを常に相手の方向に向けたりすることが格段に容易になります。
</Callout>

</Section>

<Section icon={Target} title="戦闘の進行 ＆ オンボーディングパス">

平和なロールプレイヤーから有能な校内ファイターへと転身するには、構造化された学習アプローチが必要です。ボタン連打で勝てると思ってメインの廊下へふらふらと入ってはいけません。以下のステップバイステップの進行パスに従って、マッスルメモリー（筋肉の記憶）を構築し、信頼できる戦闘習慣を確立しましょう。

<Steps>
  <Step num="1" title="設定の初期化">
    公式のRoblox体験ページからゲームを起動します。戦闘を始める前に、ゲーム内設定メニューを開き、最大フレームレートが出るようにグラフィックレベルを調整してください。シフトロックを有効にし、カメラの挙動に慣れましょう。
  </Step>
  <Step num="2" title="戦闘スタンスの切り替え">
    手を下げた状態で相手に近づいてはいけません。**R**（または対応するボタン）を押してアクティブな戦闘スタンスに入ります。この姿勢をとることで、遅延なく即座にブロック、ダッシュ、弱攻撃を実行できるようになります。
  </Step>
  <Step num="3" title="安全な間合い管理の練習">
    体育館またはワークアウトエリアに向かいましょう。**Q**でダッシュして近づき、短い弱攻撃コンボを繰り出し、すぐにダッシュで離脱する練習をします。相手を直線的に追いかけるのは避け、柱や角を利用して角度をつけ、相手のカメラトラッキングを乱しましょう。
  </Step>
  <Step num="4" title="ブロックとパリィのループをマスターする">
    スパーリングパートナーを見つけ、防御のみに集中します。**F**を長押しして攻撃をブロックし、相手の攻撃モーションの視覚的・聴覚的な合図に細心の注意を払います。攻撃が当たる瞬間にブロックを離して押し直すことで、パーフェクトパリィを発動させます。
  </Step>
  <Step num="5" title="掴みととどめを組み込む">
    相手がダウンしたら、**B**を押して危険な戦闘地帯から連れ出すか、**V**を使用して掴みフィニッシュを実行します。これにより、乱戦中に相手がチームメイトによって簡単に蘇生されるのを防ぐことができます。
  </Step>
</Steps>

<Callout type="success" title="コンボの原則">
学乱では、短く信頼性の高い連携が、過度に複雑で派手なコンボを常に上回ります。2回の弱攻撃を出し、相手の反応を観察して位置を変えることに集中してください。反撃を受けやすいフルオートコンボを出し切るのは避けましょう。
</Callout>
