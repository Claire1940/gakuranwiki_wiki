<Section icon={Swords} title="学乱ボクシング PvP：基本の戦闘メカニクス">

「学乱（Gakuran）」は2007年5月を舞台にした日本の高校生活（日常系）をテーマにしたゲームに分類されていますが、プレイヤーコミュニティの間では激しい喧嘩（対人戦）が非常に重視されています。**学乱のボクシングPvP**で勝利を収めるには、戦闘の構え、間合い管理、およびスタイルごとの相性を細かく理解する必要があります。戦闘のテンポは非常に速く、明確な戦略なしに戦いに飛び込むと、あっという間にノックダウンされてしまいます。いつ攻撃し、いつ防御すべきかを学ぶことが、この学園を制覇するための基礎となります。

現在の戦闘システムは、3つの異なる格闘スタイルを中心に構成されています。それぞれのスタイルには、プレッシャー、機動力、扱いやすさの面で独自の強みがあります。手数の多さで相手を圧倒したいのか、それとも防御を固めてカウンターを狙うのか、自分の好みのプレイスタイルに合わせて最適なスタイルを選びましょう。

<Section icon={Shield} title="デフォルトのキーバインドとデバイス操作">

校内を移動し、戦闘を有利に進めるには、デフォルトのRoblox操作設定を完全に把握しておく必要があります。学乱では、移動、カメラアングル、戦闘アクションを管理するために、標準的なキーボードショートカット、マウス入力、およびコントローラーのバインドを使用します。ゲーム内のヘルプメニューからいつでも確認できますが、命がけのPvPで生き残るためには、これらのキーバインドを体に叩き込んでおくことが不可欠です。

攻撃を当てることと同じくらい、正確な動きも重要です。回避（ダッシュ）メカニクスを駆使し、スプリントで距離を詰め、シフトロックを切り替えることで、戦況を劇的に変えることができます。モバイルやコントローラーを使用するプレイヤーは、PC版の操作感を再現するように設計された仮想ジョイスティックや割り当てられたボタンに慣れる必要があります。

<Callout type="warning" title="シフトロックに関する警告">
Robloxの設定で「シフトロック」が常に有効になっていることを確認してください。シフトロックなしで戦うと、素早く動く相手を目で追うのが極めて困難になります。
</Callout>

PCおよびコントローラーにおける全操作レイアウトについては、以下の表を参照してください。

| アクション | PC入力 | コントローラー入力 | 戦術的なアドバイス |
| **戦闘の構え** | R | 方向キー上 | 戦闘に入る前に必ず有効化 |
| **弱攻撃** | 左クリック (M1) | R2 | 素早いパンチやキックの始動技 |
| **強攻撃** | 右クリック (M2) | Y | 大ダメージを与えるが隙が大きい |
| **ガード / パリィ** | F | L2 | 長押しで受けるダメージを軽減 |
| **回避 / ダッシュ** | Q | B | コンボからの脱出や距離を詰める際に使用 |
| **走る / スプリント** | Wキー2回押し | L3 | 追跡や撤退に不可欠 |
| **トドメ / グラブ** | V | L1 + X | ダウンした相手にとどめを刺す |
| **担ぐ** | B | X | 倒れたプレイヤーを安全な場所へ移動する |

</Section>

<Section icon={Target} title="ステップバイステップ PvP上達ガイド">

初心者が学園の有能な喧嘩師へと成長するには、計画的な練習が必要です。いきなり混雑した廊下の乱闘に飛び込むのではなく、段階的なステップを踏んで上達していきましょう。これにより、基本スキルを身につけ、攻撃の隙（フレームデータ）を理解し、相手のミスを突く方法を学んだ上で、実力のあるプレイヤーに挑戦することができます。

学乱での成長は、単にスタイルを選ぶことだけではありません。状況判断力を養うことも重要です。集団戦は日常茶飯事であり、スタミナを維持しながら複数のターゲットをどう処理するかが、アマチュアとトップ層の喧嘩師を分ける境界線となります。

<Steps>
  <Step num="1" title="戦闘の構えを有効にする">
    攻撃を仕掛ける前に、必ずRキーを押して戦闘の構えに入りましょう。これにより、移動、ガード、回避が即座に行えるようになります。
  </Step>
  <Step num="2" title="間合い管理とダッシュを極める">
    Qキーを使って、相手の攻撃範囲に出入りする練習をしましょう。空振りを避けるために、自分が選んだスタイルの弱攻撃の最大リーチを把握してください。
  </Step>
  <Step num="3" title="ガードと反撃を練習する">
    Fキーを長押しして敵の攻撃をガードします。相手がコンボを終えるか、強攻撃を空振るのを待ち、即座に反撃（確定反撃）を叩き込みましょう。
  </Step>
  <Step num="4" title="短いコンボを意識する">
    長くて複雑なコンボに頼る必要はありません。確実に当たる「弱攻撃3ヒット」のコンボを繰り出し、その後は安全にダッシュで離れてポジションをリセットしましょう。
  </Step>
</Steps>

<Callout type="success" title="スタミナ管理">
ダッシュや強攻撃の連発は避けましょう。スタミナが切れると一時的に無防備になり、敵の攻撃をガードできなくなります。
</Callout>

</Section>

<Section icon={Map} title="学校マップのレイアウトとホットスポット">

ゲーム内の日本の学園都市はいくつかの主要エリアに分かれており、それぞれロールプレイ、修行、戦闘といった特定の目的を持っています。日常的な交流が楽しめる静かな教室もあれば、常に戦闘が繰り広げられている悪名高い危険地帯もあります。これらのエリアを安全に移動するには、どのルートにプレイヤー集団がたむろしているかを知る必要があります。

マップのレイアウトを理解していれば、袋小路に追い詰められるのを防げます。廊下や狭い入り口は、回避スペースが限られているため非常に危険であり、集団に囲まれやすくなります。逆に、バスケットコートのような開けた場所は、動き回るスペースが十分にあり、交戦する相手を慎重に選ぶことができます。
