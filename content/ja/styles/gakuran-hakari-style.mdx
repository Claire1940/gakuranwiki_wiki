<Section icon={Shield} title="操作方法と戦闘システム">

Robloxの操作体系をスムーズに理解していなければ、**秤（ハカリ）スタイル**をマスターすることは不可能です。学乱は標準的なRobloxエンジンを使用していますが、戦闘スタンスや高度な機動のためにカスタムキーバインドを導入しています。PC、モバイル、またはコントローラーのどれを使用しているかにかかわらず、反応速度はこれらの入力への慣れに左右されます。

最も重要なシステムは**戦闘スタンス**です。無防備な状態で歩き回ると奇襲を受けやすくなりますが、スタンス状態に入ることでキャラクターは即座にガードやダッシュができるようになります。秤スタイルの使い手にとって、「Q」キーのダッシュは最も重要な相棒であり、華麗な技の後に瞬時に位置を調整して反撃を防ぐことができます。

| アクション | PCキーバインド | モバイル入力 | コントローラー |
| :--- | :--- | :--- | :--- |
| **戦闘スタンス** | R | アイコン切り替え | 方向キー上 |
| **弱攻撃** | M1（左クリック） | 画面タップ | R2 / RT |
| **強攻撃** | M2（右クリック） | 長押し/アイコン | Y / △ |
| **ガード / パリィ** | F | シールドアイコン | L2 / LT |
| **ダッシュ / 回避** | Q | 方向スワイプ | B / 〇 |
| **処刑 / 掴み** | V | コンテキストボタン | L1 + X |
