### 深掘り：ムエタイ（攻撃の王）
ムエタイが現在の 2026 年メタを支配しているのは、前進の勢いを維持する能力があるためです。学乱では、攻撃を止めれば攻撃される可能性が高くなります。ムエタイは、パニックブロックをしている相手のガードを突き破る攻撃的なキック打撃により、このダウンタイムを最小限に抑えます。

### 深掘り：ボクシング（信頼のスタンダード）
ボクシングは依然として学乱のプレイヤー層の屋台骨です。ジムやバスケットコートで簡単に練習できるクリーンなパンチチェインを提供します。他のスタイルのような派手さには欠けますが、その根本的な強さはそのシンプルさにあります。

| スタイル | 強み | 弱点 | おすすめのプレイヤー |
| :--- | :--- | :--- | :--- |
| **ムエタイ** | 前方への圧力 / キック | 無謀な追いかけ | 攻撃的なブローラー |
| **ボクシング** | クリーンなパンチチェイン | 射程が短い | 初心者プレイヤー |
| **秤（ハカリ）** | ユニークなアイデンティティ | 習得が困難 | クリエイティブな専門家 |

<Callout type="success" title="プロの戦略">
ムエタイを使用するときは、ダッシュイン攻撃を徹底的に練習しましょう。目標は、一瞬で距離を詰め、短い反撃の窓口を作り、相手がカメラを回す前に離脱することです。
</Callout>

<Section icon={Shield} title="戦闘メカニクスと PvP ガイド">

学乱での戦闘は、ただ誰かが倒れるまでクリックするだけではありません。それはスタミナ、間合い、そして心理戦のダンスです。ベテランの不良と「練習用ダミー」の違いは、戦闘スタンスとパリィのタイミングをどう扱うかにあります。

### 戦闘スタンスの重要性
戦闘に入る前に最初に行うべきことは、戦闘スタンス（通常 PC では **R** キー）に入ることです。これにより移動プロファイルが変化し、防御オプションの準備が整います。スタンスに入らずに戦うことは、打ち合いで負けることを保証するようなものです。

### 間合い（スペーシング）と移動
間合いによって、コンボがクリーンに始まるか、人前で恥をさらす結果になるかが決まります。相手を直線的に追いかけてはいけません。代わりに、階段、入り口、角など、学校の構造物を利用して角度をつけましょう。

| 戦闘アクション | 主な目的 | 重要なコツ |
| :--- | :--- | :--- |
| **ブロック** | ダメージ軽減 | 遠くからパニックブロックしないこと |
| **パリィ** | 反撃の隙を作る | チェインの終わりにタイミングを合わせる |
| **ダッシュ** | 位置取り / 回避 | 常に背後に脱出路を確保しておく |
| **処刑** | 相手を仕留める | 周囲に敵がいない時だけ使用する |

<Callout type="warning" title="PvP の警告">
廊下や学校の入り口はリスクの高いゾーンです。出入り口は待ち伏せの「じょうご」になりやすく、グループが孤立したプレイヤーを罠にかけるために待機していることがよくあります。常にカメラをアクティブにし、360 度回転させて周囲を確認してください。
</Callout>

</Section>

<Section icon={Map} title="操作方法とキーバインドのリファレンス">

**学乱の全格闘スタイル**をマスターするには、Roblox の操作レイヤーに精通している必要があります。PC、モバイル、コントローラーのいずれであっても、これらの入力がゲームプレイの基礎となります。

| アクション | PC キーボード/マウス | モバイル（タッチ） | コントローラー |
| :--- | :--- | :--- | :--- |
| **移動** | WASD / 矢印キー | 画面上ジョイスティック | 左スティック |
| **戦闘スタンス** | R | UI ボタン | 方向キー / メニュー |
| **弱攻撃** | M1 (左クリック) | 画面タップ | R2 / RT |
| **強攻撃** | M2 (右クリック) | スペシャルアイコン | Y / △ |
| **ブロック / パリィ** | F | ガードアイコン | L2 / LT |
| **ダッシュ / 回避** | Q | ダッシュアイコン | B / 〇 |
| **スプリント** | Wを2回押し | スプリント切り替え | L3 (スティック押し込み) |
| **担ぐ / 処刑** | B / V | インタラクションポップアップ | X / L1+X |

<Callout type="tip" title="入力の最適化">
デフォルトの Roblox カメラが遅く感じる場合は、設定 > カメラモードから「クラシック」と「フォロー」を試して、動きの速いムエタイ使いをより効果的に追跡できる方を選んでください。
</Callout>

### 主要な場所とリスクレベル
- **音楽室**: PvP リスク低。交流や静かな瞬間に最適。
- **校舎**: リスク中。教室での交流は一般的ですが、数秒で乱闘が勃発することもあります。
- **街路**: リスク高。ギャングが徘徊し、大規模な戦闘が発生するオープンなテリトリーです。
- **ジム**: リスク中。主にトレーニングに使用されますが、挑戦を求める高レベルプレイヤーが頻繁に訪れます。

| 場所 | タイプ | PvP リスク | 推奨アクティビティ |

| **教室** | RPハブ | 中 | 自己紹介 / シーン作り |
| **廊下** | 通路 | 高 | 移動 / 戦闘 |
| **街** | オープンワールド | 高 | 抗争 / 徘徊 |
