<Section icon={Target} title="初心者向けオンボーディング：学園生活の始め方">

プレイヤー層の好戦的な性質もあり、初めて「学乱」に足を踏み入れると圧倒されてしまうかもしれません。このゲームは技術的には「アクション」体験に分類されますが、「日常系」の要素も含まれているため、プレイヤーはいきなり「いつどこで戦闘が始まってもおかしくない」リアルな学校環境に放り込まれます。最初の目標は、命がけの決闘に挑む前に、この環境に慣れることです。

ゲームの舞台は2007年5月の日本。この時代考証に基づいた設定は、学生服（ゲームの由来である「学ラン」）から町の建築物に至るまで、あらゆる部分に影響を与えています。新規プレイヤーは、安全地帯や訓練エリアを把握するために、まず校舎や周辺の道路を探索することに専念すると良いでしょう。

<Section icon={Shield} title="戦闘操作とキーバインドの極意">

「学乱」で活躍するためには、基本的な移動以上の操作をマスターする必要があります。このゲームは標準的なRobloxエンジンを使用していますが、戦闘の構え、ダッシュ、処刑（エグゼキューション）のためにカスタムスクリプトを実装しています。強攻撃やパリィを当てるために必要な方向の正確性を確保するには、「シフトロック（Shift Lock）」機能のマスターが必須です。

以下の表は、PC、モバイル、コントローラープレイヤー向けの必須キーバインドをまとめたものです。「戦闘の構え（Combat Stance）」（Rキー）が最も重要なボタンであることに注意してください。これを行わないと、キャラクターは「パッシブ（非戦闘）」状態のままとなり、敵の攻撃を効果的に防ぐことができません。

| アクション | PCキーバインド | モバイルジェスチャー | コントローラー |
| :--- | :--- | :--- | :--- |
| **移動** | WASD / 矢印キー | ジョイスティック | 左スティック |
| **戦闘の構え** | **R** | 画面上トグル | 方向キー上 |
| **弱攻撃** | M1 (左クリック) | 画面タップ | R2 / RT |
| **強攻撃** | M2 (右クリック) | 画面長押し | Y / 三角ボタン |
| **ブロック / パリィ** | **F** | ブロックボタン | L2 / LT |
| **ダッシュ / 回避** | **Q** | スワイプ | B / 丸ボタン |
| **処刑 / 掴み** | **V** | コンテキストボタン | L1 + X |
| **担ぐ** | **B** | コンテキストボタン | X / 四角ボタン |

<Callout type="warning" title="スタミナ管理">
すべてのダッシュや強攻撃はスタミナを消費します。戦闘中にスタミナゲージがゼロになると、「ガードクラッシュ（崩し）」状態になり、フルコンボや処刑に対して無防備になります。ダッシュで攻め込む前に、必ずゲージを確認してください。
</Callout>

</Section>

<Section icon={Map} title="マップガイド：西片高校の歩き方">

「学乱」のマップは単なる背景ではなく、戦術的な遊び場です。エリアによって危険度と得られるメリットが異なります。例えば、校舎内は交流（ロールプレイ）の場となることが多い一方、町中の道路はギャング同士のPvPが最も激化する場所です。各エリアの「危険度（Risk Level）」を理解することで、どこで特訓し、どこで交流すべきかを判断しやすくなります。

| エリア名 | タイプ | 危険度 | 特徴・見どころ |
| :--- | :--- | :--- | :--- |
| **校舎** | ロールプレイハブ | 中 | 教室、下駄箱/ロッカー、交流スポット。 |
| **体育館 / ワークアウト** | 訓練 | 中 | 移動やコンボテストの練習エリア。 |
| **町の通り** | オープンPvP | **高** | 徘徊するギャングの主要ルート、1v1の決闘。 |
| **バスケットコート** | アクティビティ | 中 | クルーの集まりによく使われる人通りの多いエリア。 |
| **音楽室** | ソーシャル | 低 | スライス・オブ・ライフ（日常系）RP用の静かなエリア。 |
| **屋上** | 決闘スポット | **高** | 突発的または計画的な1v1で人気の場所。 |

<Callout type="tip" title="マップナビゲーション">
PC版ではPキーでマップマーカーを使用し、クルーメンバーの位置を把握しましょう。どこでもPvPが発生し得るこのゲームにおいて、味方の位置を知ることは、町の通りでの待ち伏せから生き延びるための最善の方法です。
</Callout>

</Section>
