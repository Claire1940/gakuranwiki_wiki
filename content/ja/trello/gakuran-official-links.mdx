<Section icon={Target} title="スタートガイド：初心者向けオンボーディング">

初めて西片高校に入る時は、圧倒されてしまうかもしれません。従来のロールプレイゲームとは異なり、Gakuranは「オープンPvP」環境を採用しているため、最初のインタラクションが廊下での殴り合いになることもあります。体系的なオンボーディングプロセスに従うことで、最初のセッションを生き延びやすくなります。

<Steps>
  <Step num="1" title="Robloxグループに参加する">
    ゲームを起動する前に、GakuranのRobloxコミュニティページに移動しましょう。グループに参加することで、特定のゲーム内機能へのアクセスや、外部のプレイヤーには制限されているコミュニティ主導のイベントへの参加権が得られることがよくあります。
  </Step>
  <Step num="2" title="設定を構成する">
    ゲームに入ったら、Robloxの設定メニューを開きます。戦闘用にグラフィックが最適化されていることを確認してください。遅延（レイテンシ）が高かったりフレームレートが低下したりすると、テンポの速い乱闘の中でパリィ（受け流し）を行うことがほぼ不可能になります。
  </Step>
  <Step num="3" title="戦闘の構えをとる">
    身を守る方法を知らずに学校の廊下をうろついてはいけません。PCでは「R」キーを押して、すぐに戦闘の構えをとってください。これにより、キャラクターの移動やブロックの準備が整います。
  </Step>
  <Step num="4" title="トレーニングゾーンを探す">
    体育館やバスケットボールコートに向かいましょう。これらのエリアは、プレイヤーが技セットをテストするためによく使用されます。戦いに加わる前に、他のプレイヤーの戦いを見てGakuranの戦闘リズムを学びましょう。
  </Step>
</Steps>

<Callout type="tip" title="プロのヒント">
ベテランプレイヤーから常に狙われていると感じたら、「音楽室」や教室を探してみてください。これらは（常にとは限りませんが）ロールプレイヤーが集まる比較的静かなエリアであり、混沌からの一時的な避難場所になります。
</Callout>

</Section>

<Section icon={Shield} title="操作のマスター：キーバインドとメカニクス">

Gakuranは標準のRobloxエンジンを使用していますが、独自の戦闘入力レイヤーを追加しています。これらのキーバインドを理解しているかどうかが、決闘に勝つか、それとも校門まで引きずられていくかの分かれ目になります。以下の表は、PC、モバイル、およびコントローラープレイヤー向けの重要な入力をまとめたものです。

| アクション | PC キーバインド | モバイル ジェスチャー | コントローラー 入力 |
| :--- | :--- | :--- | :--- |
| **戦闘の構え** | R | 画面上ボタン | Dパッド上 |
| **弱攻撃** | M1（左クリック） | 画面タップ | R2 / RT |
| **強攻撃** | M2（右クリック） | 長押し / ボタン | Y / △ |
| **ブロック / パリィ** | F | ブロックボタン | L2 / LT |
| **ダッシュ / 回避** | Q | スワイプ / ボタン | B / 〇 |
| **スプリント** | Wの2回押し | ジョイスティックの引き伸ばし | L3（スティック押し込み） |
| **プレイヤーを担ぐ** | B | コンテキストメニュー | X / □ |
| **処刑 / 掴み** | V | コンテキストメニュー | L1 + X |

<Callout type="info">
PvPでは「シフトロック（Shift Lock）」機能を使用することを強くお勧めします。これにより、横移動しながらカメラを相手に固定できるため、攻撃を当て続けたり、パリィのタイミングを合わせたりするのが非常に容易になります。
</Callout>

戦闘システムは「スタミナ（Stamina）」と「体勢（Posture）」のバランスに依存しています。反撃せずに攻撃をブロックしすぎると、体勢が崩れて（ポスチャーブレイク）、相手の強攻撃や処刑アクションに対して無防備になります。戦闘中は常にUIバーに注意を払いましょう。

</Section>

<Section icon={Swords} title="格闘スタイル：PvPティアリストと戦闘">

Gakuranにおける格闘スタイルは、単なる見た目の違いではありません。戦闘へのアプローチ全体を決定づけるものです。一部のスタイルは高速な連撃に焦点を当てていますが、他のスタイルは重く強力な一撃を重視しています。2026年現在のメタに基づき、利用可能なスタイルをオープンPvPでの有効性ごとに分類しました。

| スタイル名 | メタ ティア | 主な強み | スキルシーリング（難易度） |
| :--- | :--- | :--- | :--- |
| **ムエタイ** | **S-Tier** | アグレッシブなプレッシャー / キック | 高い |
| **ボクシング** | **A-Tier** | バランスの取れたパンチ連携 | 普通 |
| **ハカリ** | **B-Tier** | 予測不能 / スペシャリスト | 非常に高い |
| **デフォルト** | **D-Tier** | 基本的な生存 | 低い |

<CardGrid cols={3}>
  <Card title="ムエタイ (S)">
    - **向いているプレイヤー**: 攻めの姿勢を崩さないアグレッシブなプレイヤー。
    - **キー技**: 体勢を素早く崩す膝蹴り。
    - **弱点**: パリィを避けるために正確な間合い管理が必要。
  </Card>
  <Card title="ボクシング (A)">
    - **向いているプレイヤー**: 初心者および中級者。
    - **キー技**: 相手を近づけさせない素早いジャブ。
    - **弱点**: キック主体のスタイルに比べて射程が短い。
  </Card>
  <Card title="ハカリ (B)">
    - **向いているプレイヤー**: メタに囚われないスペシャリスト。
    - **キー技**: 派手で機動性の高いカウンター。
    - **弱点**: ムエタイに比べて純粋な火力が低い。
  </Card>
</CardGrid>

<Callout type="success" title="戦闘戦略">
Gakuranで戦闘に勝つための最も効果的な方法は「後出し（リアクティブ）」で戦うことです。相手が強攻撃（M2）を外すのを待ち、ダッシュ（Q）で踏み込んで短い弱攻撃（M1）コンボを当てましょう。前に押し続けて攻撃を連打してはいけません。パリィされて手痛い反撃を受けることになります。
</Callout>

戦闘はスタイルだけでなく、環境も重要です。狭い廊下での戦闘は、狭いスペースを活かせるボクシングが有利ですが、開けた町の通り（Town Streets）では、ムエタイの優れた機動性とダッシュ踏み込みが活きてきます。

</Section>
