学乱 (Gakuran) は、日本の不良高校生たちの殺伐とした、緊張感あふれる世界へとプレイヤーを誘うユニークなRoblox体験です。もしあなたが **学乱はどんなゲームか** と疑問に思っているなら、それは本質的に「バトルグラウンド」の魂に乗っ取られた「日常」シミュレーターだと言えるでしょう。プレイヤーは象徴的な黒い詰襟の制服（名前の由来である学ラン）を身にまとい、成績ではなく拳で評判を築き上げる学校環境を突き進みます。標準的なロールプレイゲームとは異なり、学乱はメカニカルなスキル、間合いの管理、そしてスタイルの習得を重視しています。

<Section icon={Settings} title="はじめに：操作方法とキーバインド">

メカニズムの層を理解することは、**学乱はどんなゲームか** と問いかける誰にとっても最初のステップです。Robloxエンジンで動作しているため移動は馴染み深いものですが、戦闘入力は学乱独自のフレームワークに基づいています。初心者はクリッカーゲームのように扱って苦戦することが多いですが、学乱では攻撃的なプレイヤー層の中で生き残るために、意図的なブロックやダッシュの入力が必要です。

以下の表は、現在競争の激しいコミュニティの大部分を占めるPCプレイヤー向けの必須入力をまとめたものです。

| アクション | PCキーバインド | モバイル/コントローラー |
| :--- | :--- | :--- |
| **戦闘スタンス** | R | UIボタン / L3 |
| **弱攻撃** | M1 (左クリック) | 画面タップ / R2 |
| **強攻撃** | M2 (右クリック) | 画面長押し / Y |
| **ブロック / パリィ** | F | UIシールド / L2 |
| **ダッシュ / 回避** | Q | UI矢印 / B |
| **スプリント** | Wを2回押し | ジョイスティック押し込み / L3 |
| **とどめ / 掴み** | V | UIアイコン / L1 + X |
| **プレイヤーを運ぶ** | B | UIアイコン / X |

<Callout type="warning" title="スタンスの意識">
戦闘スタンス (R) に入っていないと、効果的に攻撃することができません。学校の廊下や町の通りなどの危険なゾーンに入る前には、必ずこれをオンにしてください。
</Callout>

<Steps>
  <Step num="1" title="設定の初期化">

  </Step>
  <Step num="2" title="戦闘モードに入る">

  </Step>
  <Step num="3" title="間合いのテスト">
    壁やジムでM1攻撃を使い、現在のスタイルの「当たり判定（ヒットボックス）」を理解しましょう。ボクシングはムエタイのキックよりもリーチが短いです。
  </Step>
</Steps>

<Section icon={Swords} title="戦闘メカニクスと生存術">

学乱の戦闘は単に攻撃ボタンを連打することではありません。それは「ターン」のゲームです。相手のブロックを叩いた場合、通常は相手の反撃のターンになります。「パリィ」メカニクスを理解することが、ベテランと初心者を分けるポイントです。完璧なパリィ（攻撃が当たる直前にブロック）は相手をスタンさせ、フルコンボのチャンスを作ります。

**必須の戦闘習慣：**
- **Wキーを押しっぱなしにしない**: 常に前進していると動きを読まれやすくなります。サイドダッシュ (Q + A/D) を使って相手の周囲を回りましょう。
- **スタミナに注意**: ダッシュや強攻撃はエネルギーを消費します。スタミナが切れると回避ができなくなり、格好の標的になってしまいます。
- **ヒットを確認する**: 強攻撃 (M2) を繰り出すのは、弱攻撃 (M1) を2、3回当てた後に確定させるのが基本です。

<Section icon={Map} title="マップガイド：西片高校の歩き方">

マップはいくつかの主要なゾーンに分かれており、それぞれに「リスクレベル」があります。学校が中心的なハブですが、周囲の街の通りは最も無法なPvPが発生する場所です。どこへ行くべきかを知ることは、**学乱はどんなゲームか** を理解する上で大きな部分を占めます。

| 場所 | リスクレベル | 主な目的 |
| :--- | :--- | :--- |
| **教室** | 低 | ロールプレイや集まり |
| **ジム / ワークアウト** | 中 | 練習やウォーミングアップ |
| **学校の廊下** | 高 | 待ち伏せゾーンや激戦区 |
| **街の通り** | 非常に高 | オープンワールドの抗争 |
| **屋上** | 高 | 1v1の決闘スポット（伝統的） |

<Callout type="warning" title="廊下の危険性">
廊下は狭いため、横に回避することが不可能です。ムエタイ使いに廊下で捕まると、前方へのキックによって壁際に押し込まれ、ハメられる可能性が高いです。
</Callout>

**バスケットボールコート** や **音楽室** はロールプレイコミュニティによって一般的に「中立」地帯とみなされていますが、学乱のようなゲームでは安全は決して保証されません。戦闘スタンスでこちらに向かって走ってくるプレイヤーがいないか、常にカメラを回して周囲を確認しましょう。

</Section>
