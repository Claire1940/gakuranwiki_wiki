<Section icon={Target} title="操作方法とキーバインド">

**学乱（Gakuran）ゲーム**での勝敗は、マッスルメモリー（身体に染みついた感覚）に大きく依存します。カジュアルなロールプレイゲームとは異なり、戦闘システムではブロックや回避の精密なタイミングが要求されます。ゲームは標準のRobloxエンジンを使用していますが、戦闘に参加するには手動で切り替える必要があるカスタム戦闘レイヤーが重ねられています。

以下の表は、PC、モバイル、コントローラープレイヤー向けの必須操作方法をまとめたものです。

| アクション | PCキーバインド | モバイル入力 | コントローラー |
| :--- | :--- | :--- | :--- |
| **戦闘スタンス** | **R** | 画面上のトグル | **L3 / スティック押し込み** |
| **弱攻撃（ライト）** | **M1（左クリック）** | 画面タップ | **R2 / RT** |
| **強攻撃（ヘビー）** | **M2（右クリック）** | 長押し | **Y / △ボタン** |
| **ブロック / パリィ** | **F** | シールドアイコン | **L2 / LT** |
| **回避 / ダッシュ** | **Q** | 方向スワイプ | **B / 〇ボタン** |
| **ダッシュ（スプリント）** | **Wキー2回押し** | ジョイスティックの延長 | **L3** |
| **トドメ / 掴み** | **V** | コンテキストボタン | **L1 + X** |
| **担ぐ** | **B** | コンテキストボタン | **X / □ボタン** |

<Callout type="warning" title="重要なメカニズム">
**戦闘スタンス（R）**に入っていないと、ブロックやダッシュを効果的に行うことができません。構えずに歩き回っていると、熟練プレイヤーにとって格好の「無防備な標的」になってしまいます。
</Callout>

移動は打撃と同じくらい重要です。プレイヤーがよく犯すミスは「Wキーを押しっぱなしにする（常に前進する）」ことで、これによって攻撃が予測されやすくなります。優秀な喧嘩師は**Qキー**を使って横方向にダッシュし、相手のブロックをかいくぐる角度を作り出します。

</Section>

<Section icon={Shield} title="戦闘ガイド：初心者から一人前の不良へ">

**学乱（Gakuran）ゲーム**での初めての戦闘は、圧倒されるかもしれません。「スタン（怯み）」メカニズムがあるため、一度攻撃を食らうと、「エスケープ」や「カウンター・パリィ」の方法を知らない限り、さらに3回連続で殴られることになります。廊下での勝率を上げるために、以下のステップを実践しましょう。

<Steps>
  <Step num="1" title="「パーフェクトパリィ」をマスターする">
    単に**Fキー**を長押しするだけではいけません。相手の腕が前に動く瞬間を待ち、Fキーをタップします。パリィが成功すると相手がスタンし、フルコンボを叩き込む絶好のチャンスが生まれます。
  </Step>
  <Step num="2" title="スタミナを管理する">
    ダッシュ、回避、強攻撃はスタミナを消費します。スタミナゲージがゼロになると移動速度が低下し、ブロックもできなくなります。スタミナが25%を下回ったら、必ず一度引きましょう。
  </Step>
  <Step num="3" title="「M1-M1-M1-M2」コンボを覚える">
    最も基本でありながら効果的な連携は、弱攻撃を3回繰り出した後に強攻撃を繋げる方法です。相手が弱攻撃をブロックしている場合、最後の強攻撃が「ガードブレイク」として機能します。
  </Step>
  <Step num="4" title="周囲の環境を利用する">
    **学乱（Gakuran）ゲーム**では、ロッカーや壁にプレイヤーを追い詰める（壁ハメする）ことができます。これにより相手の回避ルートが制限され、「掴み（Grab）」（Vキー）のフィニッシャーを当てやすくなります。
  </Step>
</Steps>

<Callout type="warning" title="「Wキー連打」を避ける">
上級者はあなたが突進してくるのを待ち構え、タイミングよくパリィキーを押してきます。後退したり横ダッシュを混ぜたりして、相手に早めのブロックを誘発させるなど、動きに変化をつけましょう。
</Callout>

</Section>

<Section icon={Map} title="マップ探索とロールプレイ（RP）の重要スポット">

**学乱（Gakuran）ゲーム**のマップは、日本の町並みを細部まで再現した緻密な設計になっています。PvPは至る所で発生しますが、特定の場所はコミュニティ内で特別な目的を持っています。行くべき場所を知っておくことで、練習相手を見つけたり、「ギャンカー」（ソロプレイヤーを集団で襲うグループ）を回避したりできます。

| ロケーション | 危険度 | 目的・用途 |
| :--- | :--- | :--- |
| **教室** | **低** | 主にソーシャルロールプレイや、クルー（仲間）との待ち合わせに使用。 |
| **学校の廊下** | **高** | まさに「戦場」。ロッカー周辺では常に戦闘や不意打ちが発生。 |
| **体育館 / トレーニングエリア** | **中** | 動きの練習や、新しい戦闘スタイルのテストを行う場所。 |
| **街の通り** | **高** | オープンワールドPvP。大規模な「ギャング」たちが縄張り争いを繰り広げる。 |
| **音楽室** | **低** | 日常（Slice of Life）的な要素を好むプレイヤー向けの静かなスポット。 |

**学乱（Gakuran）ゲーム**が人気を博している最大の理由は、その「雰囲気」にあります。たとえ戦う気がなくても、夜の街（「ネオンレッド」の美学）を探索することは、このゲームの重要な魅力の一部です。多くのプレイヤーが、2007年当時の不良の雰囲気に合わせてキャラクターの見た目をカスタマイズすることだけに、何時間も費やしています。

<Callout type="info" title="探索のコツ">
入り組んだ街の通りで迷子になった場合は、**Pキー**（マップマーカー）を使って目的地への目印を設定しましょう。
</Callout>

</Section>
