<Section icon={User} title="学乱でのヘッドホンの付け方とカスタマイズ">

自己表現（キャラクターメイク）は、学乱のゲーム体験における重要な柱です。「一匹狼」の不良スタイルを目指すにしても、現代的な学生の美学を追求するにしても、**学乱でのヘッドホンの付け方**やアクセサリーの管理方法を知っておくことは不可欠です。Robloxの学乱では、アクセサリーの着用は通常、Roblox公式のアバターエディターと、開発者が提供するゲーム内の特定カスタマイズメニューを組み合わせて行われます。

ヘッドホンを装備するには、一般的に2つの方法があります。ご自身のRobloxインベントリ（所持品）から恒久的に装備するか、特定のサーバーバージョンがローカルな見た目の上書きに対応している場合はゲーム内の「編集（Edit）」機能を利用します。多くのプレイヤーは、特定のロールプレイのペルソナ（多くの場合、「寡黙だが腕の立つファイター」）を表現するために、「ヘッドホンをかけた不良」スタイルを好みます。

<CardGrid cols={3}>
  <Card title="アバターエディター方式">
    - **恒久的な見た目**: Robloxウェブサイトのエディターでヘッドホンを装備します。
    - **互換性**: すべての学乱サーバーで動作します。
    - **多様性**: インベントリに所有している任意のヘッドホンを使用できます。
  </Card>
  <Card title="ゲーム内メニュー方式">
    - **動的な変更**: カスタマイズUI（有効な場合）にアクセスします。
    - **ロールプレイ重視**: 現在の学校のシチュエーションに合わせて見た目を変えられます。
    - **クイック切り替え**: 戦闘用スタイルとカジュアルな服装を素早く切り替えられます。
  </Card>
  <Card title="アクセサリーの重ね着">
    - **高度なスタイル**: ヘッドホンと帽子やマスクを組み合わせます。
    - **干渉チェック**: 学ランの立ち襟と重なってグラフィックが崩れない（クリッピングしない）か確認します。
    - **ビジュアルアイデンティティ**: 自分のグループ（チーム）専用のユニークな「ギャング制服」を作成できます。
  </Card>
</CardGrid>

<Callout type="warning" title="グラフィックの干渉（クリッピング）について">
学ランのジャケットは襟が高く硬いため、一部の大型ヘッドホンモデルはキャラクターのメッシュを突き抜けて表示（クリッピング）されることがあります。最高の見た目を実現するために、小さめの「イヤホン」タイプのアクセサリーや、スリムな形状のヘッドホンを試してみてください。
</Callout>

<Section icon={Target} title="キャラクターセットアップのステップ・バイ・ステップ">

このゲームを初めてプレイする場合、キャラクターのセットアップは見た目を変えるだけにとどまりません。体育館でノックアウトされている最中におしゃれなヘッドホンが落ちてしまわないよう、操作設定が正しく設定されているか確認する必要があります。2026年版の学乱を始めるために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="公式コミュニティへの参加">
    ゲームを起動する前に、「(学乱) Gakuran」の公式Robloxグループに参加しましょう。これにより、特定のコミュニティ機能へのアクセスが許可されたり、ゲストではなく「正式な生徒」として認識されたりすることがあります。
  </Step>
  <Step num="2" title="Robloxの設定を調整する">
    Robloxの設定を開き、**シフトロック（Shift Lock）**を有効にします。これは学乱の戦闘において必須の設定であり、テンポの速い攻防の中でキャラクターの向きを固定（フォーカス）し続けることができます。
  </Step>
  <Step num="3" title="カスタマイズハブにアクセスする">
    ゲーム内に入ったら、カスタマイズのアイコンまたはエリア（通常はスポーン地点の近く、またはメインメニュー内）を探します。ここで、UIを通じて**学乱でのヘッドホンの付け方**の疑問が解決します。
  </Step>
  <Step num="4" title="戦闘スタイルを選択する">
    トレーニングエリアに向かい、初期スタイルを選択します。ボクシングであれムエタイであれ、選択したスタイルによって移動モーションや、キャラクターの動きに合わせたアクセサリーの揺れ方が決まります。
  </Step>
</Steps>

<Callout type="success" title="戦闘準備完了">
キャラクターのカスタマイズと操作設定が完了すれば、西方高校を探索する準備は万端です。ロールプレイ要素の強い格闘ゲームにおいて、「見た目を整えること」はバトルの半分を占める重要な要素であることを忘れないでください。
</Callout>

</Section>

<Section icon={Settings} title="操作方法とキーバインドのマスター">

生き残るためには、操作方法を理解することが極めて重要です。**学乱でのヘッドホンの付け方**に気を取られているかもしれませんが、ブロックやパリィのやり方を知らなければ、すぐにやられてしまいます。学乱は標準的なRobloxエンジンを使用していますが、その戦闘システムには独自のレイヤーが追加されています。

| アクション | PC版キーバインド | モバイル / コントローラー | 備考 |
| :--- | :--- | :--- | :--- |
| **移動** | WASD / 矢印キー | ジョイスティック / 左スティック | 校内を移動するための基本操作。 |
| **戦闘構え** | **R** | UIボタン | 攻撃を繰り出すために有効化する必要があります。 |
| **弱攻撃** | **M1 (クリック)** | 画面タップ / R2 | 出が早くダメージが低いコンボ始動技。 |
| **強攻撃** | **M2 (右クリック)** | UIボタン / Y | ガード崩し（シールドブレイカー）。使用後の隙（硬直）が大きいです。 |
| **ブロック / パリィ** | **F** | UIボタン / L2 | 受けるダメージを無効化するために不可欠な操作。 |
| **ダッシュ / 回避** | **Q** | UIボタン / B | 位置取りや攻撃の回避に使用します。 |
| **とどめ / 掴み** | **V** | UIボタン / L1 + X | ダウンした相手に対するフィニッシャー（処刑）。 |

<Callout type="info" title="入力遅延（インプットラグ）対策のヒント">
M1（弱攻撃）の連撃に遅延を感じる場合は、グラフィック設定を確認してください。Robloxのグラフィック品質を下げることで、パリィやダッシュの反応速度が大幅に向上することがあります。
</Callout>

</Section>
