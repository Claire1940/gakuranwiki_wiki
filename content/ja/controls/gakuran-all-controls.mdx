<Section icon={Map} title="コアな移動とインタラクトのキーバインド">

**学乱 (Gakuran) の全操作**を理解するには、まず基本のナビゲーションから始まります。学乱は2007年の日本の学校環境を詳細に再現した舞台であるため、移動は単なる歩行以上の意味を持ちます。狭い廊下や広い校庭でのキャラクターの立ち振る舞いを管理する必要があります。ベースは標準のRobloxエンジンですが、開発者によって「不良」ロールプレイ体験を高めるための特定のレイヤーが追加されています。

| アクション | PCキーバインド | モバイル入力 | コントローラー入力 |
| :--- | :--- | :--- | :--- |
| **歩行/移動** | WASD / 矢印キー | ジョイスティック | 左スティック |
| **スプリント** | Wを2回押し | スプリントボタン | L3 (スティック押し込み) |
| **ジャンプ** | スペースキー | ジャンプアイコン | A / ×ボタン |
| **カメラ** | 右クリックドラッグ | 画面スワイプ | 右スティック |
| **インタラクト** | E / クリック | タッププロンプト | X / □ボタン |
| **チャット** | / キー | チャットアイコン | メニューナビゲーション |
| **マップマーカー** | P キー | UIボタン | 十字キー下 |

<Callout type="tip" title="プロのヒント：シフトロック">
Robloxの設定メニューで**シフトロック (Shift Lock)** を有効にすると、カメラをキャラクターの背後に固定できます。これは、高速な校内乱闘中に常に「前」を向き続けるために不可欠であり、ベテランプレイヤーにとっては必須の設定です。
</Callout>

適切な移動ができるかどうかで、うまく逃げ切れるか、音楽室の隅に追い詰められるかが決まります。スプリント中はキャラクターの姿勢に注目してください。ゲームにはカスタムアニメーションが採用されており、他のプレイヤーに自分の意図を伝える役割も果たします。社交的な場面では、歩行（Shiftキーを押し続けるか、コントローラーを軽く傾ける）を使用することで、より没入感のあるロールプレイが可能です。

</Section>

<Section icon={Swords} title="戦闘メカニクスと格闘スタイル">

学乱 (Gakuran) 体験の核心は、バトルグラウンド（Battlegrounds）にインスパイアされた戦闘システムにあります。標準的なRPGとは異なり、このゲームではすべての打撃とパリィに能動的な入力が必要です。戦闘を開始するには、まず**戦闘スタンス**を切り替える必要があります。このスタンスが有効でないと、ブロックや特殊コンボの実行ができず、廊下で「ランダマー（通りがかりの攻撃者）」に対して無防備になってしまいます。

<CardGrid cols={3}>
  <Card title="ムエタイ (Muay Thai) スタイル">

    - **最適**: 絶え間ない攻撃を好む近接戦闘狂向け。
    - **難易度**: 普通。
  </Card>
  <Card title="ボクシング (Boxing) スタイル">

    - **難易度**: 初級。
  </Card>
  <Card title="秤 (Hakari) スタイル">
    - **テクニカルな演出**: 独特のタイミングを持つ派手な動き。

    - **難易度**: 上級。
  </Card>
</CardGrid>

<Callout type="warning" title="戦闘の警告">
攻撃を「連打（スパム）」しないでください。学乱では忍耐が報われます。強攻撃（M2）を外すと、リカバリーアニメーションで動けなくなり、相手にフルコンボを叩き込まれる隙を与えてしまいます。
</Callout>

| 戦闘アクション | PCキーバインド | 効果 / 用途 |
| :--- | :--- | :--- |
| **戦闘スタンス** | R | 戦闘モードの切り替え。ダッシュとブロックが可能になる。 |
| **弱攻撃** | M1 (左クリック) | コンボを組み立てるための、速くて低ダメージの打撃。 |
| **強攻撃** | M2 (右クリック) | 高ダメージ。タイミングが合えばガードを崩す。 |
| **ブロック / パリィ** | F | 受けるダメージを軽減。タイミングよくブロックするとパリィになる。 |
| **回避 / ダッシュ** | Q | 強攻撃を避けるための素早い方向移動。 |
| **プレイヤーを運ぶ** | B | ダウンした相手を拾い上げ、別の場所へ運ぶ。 |
| **処刑 / 掴み** | V | 戦闘を終わらせるか、グラップリング技を開始する。 |

**M1, M1, M1, M2** のリズムをマスターするのが、ほとんどの格闘スタイルにおける標準的な基本コンボです。しかし、真のスキルは、迫りくる強攻撃をパリィするためにいつ **F** を押すべきかを知ることにあります。パリィに成功すると攻撃者は短時間スタンし、壊滅的なカウンター攻撃のチャンスが生まれます。

<Section icon={Target} title="ステップバイステップ：最初の喧嘩をマスターする">

学校に来たばかりの頃は、**学乱 (Gakuran) の全操作**が圧倒的に感じられるかもしれません。混乱した生徒から尊敬される喧嘩師へと成長するために、以下の手順に従ってください。危険な街の通りへ向かう前に、体育館やワークアウトエリアでこれらのステップを練習しましょう。

<Steps>
  <Step num="1" title="戦闘スタンスの初期化">
    脅威を感じたらすぐに **R** を押してください。キャラクターが戦闘の構えをとります。これにより、**Q (ダッシュ)** と **F (ブロック)** の入力が即座に使えるようになります。
  </Step>
  <Step num="2" title="間合いの管理">
    移動キーを使用して相手の周りを回ります。単に **W** を押し続けて突っ込むのはやめましょう。相手が強攻撃の予備動作に入ったら、**Q** を使って横にダッシュします。
  </Step>
  <Step num="3" title="M1で始動">
    弱攻撃 (M1) でコンボを開始します。ヒットしたらそのままコンボを繋げます。相手がブロックした場合はコンボを止め、ダッシュで離れるか相手の反撃をブロックする準備をします。
  </Step>
  <Step num="4" title="仕上げまたは移動">
    相手がダウンしたら選択肢があります。戦闘を終わらせるなら **V** で処刑し、さらなるロールプレイのため、あるいは廊下を空けるために別の場所へ運ぶなら **B** を使用します。
  </Step>
</Steps>

<Callout type="success" title="成功のヒント：スタミナループ">
隠されたスタミナ値に注意してください。ダッシュや強攻撃を繰り返すと、最終的に動作が遅くなります。攻撃の出が遅くなったと感じたら、一旦下がり、**F** を押し続けて防御しながら回復を待ちましょう。
</Callout>

</Section>

<Section icon={Shield} title="プラットフォーム別の詳細とモバイルの最適化">

PC版が最も細かなコントロールを提供しますが、学乱はアクセシビリティを考慮して設計されています。モバイルプレイヤーには、PCの操作を反映した専用のUIオーバーレイが用意されています。「戦闘スタンス」ボタンは通常右下に配置されており、学校生活の合間に素早く切り替えることができます。

| 機能 | PC体験 | モバイル体験 | コントローラー体験 |
| :--- | :--- | :--- | :--- |
| **精度** | 最高 (マウスエイム) | 中程度 (オートロック) | 高い (アナログ移動) |
| **コンボ速度** | 速い (メカニカルキー) | 変動あり (タッチ遅延) | 速い (トリガー) |
| **UIの明瞭さ** | 清潔 (キーバインド) | 煩雑 (画面上ボタン) | 清潔 (動的プロンプト) |
| **利用可能性** | 全機能利用可能 | 全機能利用可能 | 全機能利用可能 |

<Callout type="info" title="コントローラーサポート">
コントローラーを使用する場合、ゲームは **L2** をブロック、**R2** を弱攻撃に割り当てています。この設定は現代の格闘ゲームに似ており、Robloxに参加するコンソールプレイヤーにとっても直感的です。
</Callout>

モバイルプレイヤーは、複雑なパリィのタイミングよりもポジショニングに集中すべきです。Robloxの設定で「ロックオン」機能が利用可能な場合は、それを使用して相手を常に視界に入れておきましょう。PCプレイヤーは、マウスの感度を180度回転できる程度に高く設定してください。これは、学校の狭い階段で複数の不良と戦う際に不可欠です。

</Section>
