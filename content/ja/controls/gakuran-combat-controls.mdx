<Callout type="info" title="クイックガイド">
- **主要な操作**: **学乱の戦闘操作**をマスターするには、まず **R** キーで戦闘態勢（Combat Stance）を切り替えることから始めましょう。
- **基本キーバインド**: **M1**（左クリック）で弱攻撃、**M2**（右クリック）で強攻撃を繰り出し、**F** でガードを行います。

- **生存のコツ**: **Q** キーのダッシュで常に移動し続け、リスクの高い校内の廊下で追い詰められないようにしましょう。

<Section icon={Swords} title="学乱の戦闘操作とデフォルトのキーバインド">

最初の遭遇で生き残るためには、デフォルトの**学乱の戦闘操作**を体に叩き込む必要があります。このゲームはRobloxエンジン上で動作しており、プレイヤーが手動で戦闘準備を切り替える必要があるカスタム戦闘レイヤーを利用しています。

最も重要な入力は「戦闘態勢（Combat Stance）」の切り替えです。態勢をアクティブにしない限り、キャラクターは攻撃の実行、飛んでくる打撃のブロック、高度な回避行動を行うことができません。

| アクション | キーバインド | 戦術的用途 |
| :--- | :--- | :--- |
| 戦闘態勢 | R | ガードを構え、攻撃入力を有効にする |
| 弱攻撃 | M1 / 左クリック | 出が速く、コンボの起点に最適 |
| 強攻撃 | M2 / 右クリック | 動作は遅いが威力が高く、ガードを崩せる |
| ブロック / パリィ | F | タイミングを合わせることで基本攻撃を無効化する |
| 回避 / ダッシュ | Q | 素早く位置を変え、強攻撃を回避する |
| スプリント | Wを2回押し | 距離を詰めたり、乱闘から離脱したりする際に使用 |
| 担ぐ | B | ダウンした相手を安全な場所へ運ぶ |
| とどめ / 掴み | V | 倒した相手を排除してポイントを獲得する |
| マップマーカー | P | クルーと連携するために場所をマークする |

他のプラットフォームを使用するプレイヤーのために、学乱はコントローラーやタッチスクリーンに適した入力に変換されています。正確なタイミングが要求されるためPCが好まれますが、適切なレイアウト調整を行えばモバイルやコンソールでも十分にプレイ可能です。

| アクション | コントローラー入力 | モバイル入力 |
| :--- | :--- | :--- |
| 移動 | 左スティック | 画面上のジョイスティック |
| ジャンプ | A / ×ボタン | ジャンプボタン |
| カメラ | 右スティック | 画面ドラッグ |
| 攻撃 | R2トリガー | 攻撃アイコンをタップ |
| ブロック | L2トリガー | シールドアイコンをタップ |
| ダッシュ | B / 〇ボタン | ダッシュアイコンをタップ |

<Callout type="tip" title="シフトロックの最適化">
Robloxの設定メニューで「Shift Lock」を有効にしましょう。これによりカメラがキャラクターの背後に固定され、攻撃のエイムや動くターゲットの追跡が格段に容易になります。
</Callout>

</Section>

<Section icon={Target} title="不可欠な戦闘メカニクスとPvP戦略">

学乱のPvPで成功するには、間合い（スペーシング）、ガードの規律、そしてスタイルの相性を理解することが不可欠です。ボタンを連打しているだけでは、空振りを逃さず反撃してくる経験豊富なプレイヤーにすぐに敗北してしまいます。

選択する格闘スタイルによって、技の構成、射程、全体的なテンポが決まります。現在、コミュニティでは戦術的な好みに合わせて3つの異なるスタイルが利用されています。

<CardGrid cols={3}>
  <Card title="ムエタイ (Sランク)">
    - **攻撃的な打撃**: 強力な蹴り主体の近接攻撃を繰り出す。
    - **高いプレッシャー**: 相手を追い詰め、ミスを誘うのに最適。
    - **間合い依存**: パリィを避けるために正確なスペーシングが必要。
  </Card>
  <Card title="ボクシング (Aランク)">
    - **初心者向け**: シンプルで読みやすいパンチコンボで、習得が容易。
    - **安定した殴り合い**: 近距離で着実にダメージを与える。
    - **防御重視**: 活躍するには堅実なガードの規律が必要。
  </Card>
  <Card title="ハカリ (Bランク)">
    - **華麗なスペシャリスト**: 独特な視覚効果と変則的な技が特徴。
    - **意表を突く要素**: 標準的なスタイルを予想している相手を釣るのに適している。
    - **高い難易度**: 効果的に使いこなすには、強い相性知識が求められる。
  </Card>
</CardGrid>

安定して勝つためには、「釣り（ベイティング）」の技術をマスターしなければなりません。相手の攻撃範囲内に一瞬入り、回避キーですぐに後ろにダッシュし、相手が攻撃後の硬直状態にある間に弱攻撃コンボで即座に攻め込みましょう。

<Callout type="success" title="コンボの鉄則">
長く複雑な連携よりも、短く確実なコンボを重視しましょう。短い連携は相手にパリィされにくく、状況が悪化した場合にダッシュで離脱するためのスタミナを残しておくことができます。
</Callout>

</Section>

<Section icon={Sparkles} title="初心者向け戦闘上達ステップ">

学校での乱闘に慣れていない場合、いきなり混雑した場所に飛び込むとフラストレーションが溜まるだけです。以下の段階的な上達パスに従って、安全にスキルを磨きましょう。

<Steps>
  <Step num="1" title="クライアント設定の構成">
    学乱にログインし、Robloxの設定を開いてカメラモードを「Classic」に設定します。Shift Lockが有効であることを確認し、激しい戦闘中でもフレームレートを安定させるためにグラフィックス設定を調整します。
  </Step>
  <Step num="2" title="戦闘態勢切り替えの練習">
    マップの静かな場所を見つけ、**R** キーを押して戦闘態勢に入る練習をします。切り替えの速度に慣れ、相手が近づいてきた瞬間に態勢を整えられるようにしましょう。
  </Step>
  <Step num="3" title="ブロックとダッシュのタイミングをマスターする">
    友人やプレッシャーの少ない相手とスパーリングをします。**F** を押し続けて相手の弱攻撃をブロックすることに集中し、相手が強攻撃の予備動作を見せた瞬間に **Q** を押して横にダッシュする練習をします。
  </Step>
  <Step num="4" title="基本的な反撃（パニッシュ）の実行">
    相手の攻撃をブロックまたは回避できたら、即座に3ヒットの弱攻撃コンボで追撃します。深追いせず、3ヒット目が終わったら一度位置をリセットしましょう。
  </Step>
</Steps>

<Callout type="info" title="スタミナ管理">
ダッシュ、ブロック、攻撃はすべてスタミナを消費します。スタミナバーが完全に空になるとガードが崩れ（ガードブレイク）、数秒間スタン状態になり無防備になってしまいます。
</Callout>

</Section>

<Section icon={Shield} title="マップナビゲーションと危険な廊下地帯">

学乱のマップは、穏やかなソーシャル・ロールプレイ環境と、混沌としたPvPスポットに分かれています [2]。キャンパス内のどこにいるかによって、潜在的な脅威に対してどれだけ速く反応すべきかが決まります。

狭い出入り口や廊下などのチョークポイントは非常に危険です。簡単に追い詰められ、移動の選択肢が制限されてしまうからです。

| 場所 | PvPリスク | 主な目的 | 戦術的メモ |
| :--- | :--- | :--- | :--- |
| 校舎 | 中 | ソーシャル・ロールプレイと探索 | 教室は大人数からの避難場所として有効 |
| 廊下 | 高 | 主要な動線とチョークポイント | 立ち止まるのは厳禁。出入り口は待ち伏せされやすい |
| バスケットボールコート | 中 | カジュアルな溜まり場と野外スパーリング | 公平なデュエルを望むプレイヤーを見つけやすい |
| ジム | 中 | トレーニングと動きの練習 | ダッシュやブロックのタイミングを温めるのに最適 |
| 街路 | 高 | オープンPvPとギャング抗争 | 組織化されたグループに数で圧倒されるリスクが高い |
