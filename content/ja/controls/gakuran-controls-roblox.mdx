<Section icon={Laptop} title="PC版：学乱 (Gakuran) の必須操作方法">

**学乱 (Gakuran) の操作方法** を理解することは、不良たちがたむろする西片高校の廊下で生き残るための第一歩です。標準的なロールプレイ体験とは異なり、学乱は正確なタイミングとキーボードレイアウトへの習熟が必要な戦闘重視のシステムを採用しています。移動システムは標準的なRobloxの枠組みに従っていますが、スプリントやスタンス管理によって深みが増しています。

| アクション | キー設定 | 備考 |
| :--- | :--- | :--- |
| **移動** | WASD / 矢印キー | 標準的な方向移動 |
| **ジャンプ** | スペース | 移動や空中復帰に使用 |
| **スプリント** | Wを2回押し | 距離を詰めたり逃げたりするのに必須 |
| **戦闘スタンス** | R | 戦闘準備状態の切り替え |
| **シフトロック** | Shift | 戦闘の精度を高めるためにカメラを固定 |
| **インタラクト** | E | ドア、NPC、アイテムに使用 |
| **チャット** | / | コミュニケーションウィンドウを開く |
| **メニュー** | ESC | 設定やヘルプインターフェースにアクセス |

<Callout type="warning" title="スタンスの必要性">
「パッシブ」モードの間は効果的に攻撃できません。校門や広場などの人通りの多いエリアに入る前に、必ず戦闘スタンス（R）がオンになっていることを確認してください。
</Callout>

単純な移動だけでなく、カメラとキャラクターの相互作用も重要です。PvPではシフトロックの使用を強く推奨します。これにより、相手の動きを注視しながら周囲を回るように動く（サークリング）ことが可能になります。これは、カジュアルプレイヤーと上位ランカーを分ける **学乱 (Gakuran) の操作方法** の核心的な要素です。

</Section>

<Section icon={Swords} title="コンバットキー設定と高度なメカニクス">

学乱の戦闘は、弱攻撃、強攻撃、ブロックによる「じゃんけん」のようなメカニクスを中心に構成されています。コミュニティが「バトルグラウンド」系のサブジャンルに大きく傾倒しているため、これらの **学乱 (Gakuran) の操作方法** をマスターすることは不可欠です。すべての入力には特定のリカバリー（硬直）時間があるため、無謀なボタン連打はすぐに敗北を招きます。

| 戦闘アクション | PC入力 | モバイル / コントローラー |
| :--- | :--- | :--- |
| **弱攻撃** | 左クリック (M1) | 画面タップ / R2 |
| **強攻撃** | 右クリック (M2) | 強攻撃アイコン / Y |
| **ブロック / パリィ** | F | 盾アイコン / L2 |
| **回避 / ダッシュ** | Q | ダッシュアイコン / B |
| **とどめ / 掴み** | V | ドクロアイコン / L1 + X |
| **担ぐ** | B | 手のアイコン / X |
| **マップマーカー** | P | UIボタン |

<Callout type="tip" title="パリィのタイミング">
相手の攻撃が当たる瞬間にブロックキー（F）をタップするとパリィが発動し、相手を反撃に対して無防備な状態にします。強攻撃でガードを崩される可能性があるため、無闇にブロックを押し続けないでください。
</Callout>

上級プレイヤーは、ダッシュ（Q）を回避だけでなく「ダッシュダンス」にも利用します。これは、相手の間合いにダッシュで入り込んで攻撃を誘い、即座にダッシュで離れて空振りを狩る（パニッシュ）テクニックです。この戦術的なレイヤーにより、**学乱 (Gakuran) の操作方法** システムは見かけ以上に奥深いものになっています。

</Section>

<Section icon={Target} title="PvP初心者向けステップバイステップガイド">

学乱を始めたばかりでいきなり戦いに飛び込むと、すぐにリスポーン画面に戻ることになるでしょう。以下の手順に従って **学乱 (Gakuran) の操作方法** を調整し、トップランクの不良たちに挑む前に筋肉の記憶（マッスルメモリー）を構築してください。

<Steps>
  <Step num="1" title="設定の調整">
    Robloxメニュー（ESC）を開き、「設定」に移動します。「シフトロックスイッチ」をオンにし、安定した60 FPSを確保できるレベルにグラフィックスを設定してください。ラグは戦闘精度の最大の敵です。
  </Step>
  <Step num="2" title="基本コンボの練習">
    ジムや街の通りの静かな場所を見つけます。スタンス（R）を切り替え、M1, M1, M1, M2のコンボを練習しましょう。これがほとんどのスタイルにおける基本のコンボです。
  </Step>
  <Step num="3" title="ダッシュキャンセルの習得">
    攻撃した直後にQを押してダッシュで離れる練習をします。この「ヒット・アンド・アウェイ」戦術は、複数のプレイヤーに狙われる可能性がある集団戦を生き抜くために不可欠です。
  </Step>
  <Step num="4" title="フレンドリーなプレイヤーとデュエル">
    バスケットボールコートの近くで、リスクの低いデュエルに参加しましょう。攻撃するよりも、ブロック（F）とパリィに完全に集中してください。殺し方を学ぶよりも、生き残り方を学ぶことの方が重要です。
  </Step>
</Steps>

<Callout type="success" title="プロの戦略">
多くのプレイヤーは体力が低くなるとパニックダッシュをします。相手がダッシュを使い切るまで自分のダッシュを温存し、その後ダッシュで距離を詰めて最後の一撃（強攻撃）を叩き込みましょう。
</Callout>

</Section>
