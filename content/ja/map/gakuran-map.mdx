<Section icon={Swords} title="基本操作と戦闘メカニクス">

学乱（Gakuran）の戦闘は非常にテンポが速く、間合い（スペーシング）、タイミング、そして防御の規律が極めて重要です。ゲームはRobloxのコアエンジン上で動作しているため、オープンワールドでの小競り合いを生き延びるための第一歩は、デフォルトの操作方法をマスターすることです。攻撃を繰り出したり、敵の攻撃を防いだりする前に、必ず「戦闘構え（Combat Stance）」に入る必要があります。

以下の表は、PC、モバイル、およびコントローラープレイヤー向けのデフォルト操作をまとめたものです。

| アクション | キーボード＆マウス | モバイルジェスチャー | コントローラー入力 |
| :--- | :--- | :--- | :--- |
| 移動 | WASD / 矢印キー | 画面上ジョイスティック | 左スティック |
| ジャンプ | スペースキー | 画面上ジャンプボタン | Aボタン |
| カメラ操作 | 右クリックドラッグ | 画面スワイプ | 右スティック |
| 戦闘構え | Rキー | ゲーム内メニューオプション | ヘルプメニューショートカット |
| 弱攻撃 | 左クリック (M1) | 画面タップ | R2トリガー |
| 強攻撃 | 右クリック (M2) | 画面タップ | Yボタン |
| ガード / パリィ | Fキー | 画面上ガードアイコン | L2トリガー |
| 回避 / ダッシュ | Qキー | 画面上ダッシュアイコン | Bボタン |
| 相手を担ぐ | Bキー | タッチプロンプト | Xボタン |
| とどめ / 掴み | Vキー | タッチプロンプト | L1 + Xボタン |

戦闘で安定して勝利を収めるためには、これらの基本的な操作を体系的なパターンとして組み合わせる必要があります。闇雲に攻撃を振り回すのは避けましょう。相手が強攻撃を外すのを待つか、相手の攻撃をガードしてから、素早い弱攻撃のコンボを叩き込むのが基本です。

<Callout type="warning" title="戦闘時の注意">
ガードボタンを押しっぱなしにしないでください。上級者プレイヤーは強攻撃で簡単にガードを崩してきます。ガードが崩されると気絶（スタン）状態になり、大ダメージコンボをまともに受けてしまいます。
</Callout>

</Section>

<Section icon={Sparkles} title="戦闘スタイル＆PvPティアリスト">

選択する戦闘スタイル（格闘スタイル）によって、攻撃速度、前方へのプレッシャー、コンボの可能性など、戦闘ツールキットのすべてが決まります。「学乱（Gakuran）」には、好戦的な近接アタッカーから防御的なカウンターパンチャーまで、さまざまなプレイスタイルに対応した独自のスタイルが存在します。自分に合ったスタイルを選ぶことは、個人の使いやすさだけでなく、集団戦での役割分担においても非常に重要です。

現在の戦闘スタイルを、汎用性、使いやすさ、そして開かれた学校内でのPvPにおける総合的な強さに基づいてランク付けしました。

| スタイル | ティア | 主な役割 | 強み | 練習のポイント |
| :--- | :--- | :--- | :--- | :--- |
| ムエタイ（Muay Thai） | S | 攻撃的ストライカー | 強力な前方へのプレッシャー、キックによる反撃 | ダッシュ攻撃とブロックのタイミング |
| ボクシング（Boxing） | A | バランス型パンチャー | 初心者向け、クリーンなパンチ連携 | 安全な離脱ルートと間合いの管理 |
| ハカリ（Hakari） | B | 特殊系ブローラー | 独特なビジュアル、高い意表を突く性能 | ガードを誘う立ち回りとカウンター |
