<Section icon={Zap} title="操作方法とキーバインドの習得">

西片高校の廊下で生き残るためには、操作を無意識に行えるようになる必要があります。このゲームは、移動と高強度の戦闘アクションを分けた階層的なコントロールスキームを採用しています。PCプレイヤーは一般的に精度において有利ですが、2026年のアップデートでモバイルインターフェースも大幅に最適化されました。

| アクションタイプ | PCキーバインド | モバイル/コントローラー | 戦闘での用途 |
| :--- | :--- | : :--- | :--- |
| **戦闘スタンス** | **R** | UIトグル / R3 | すべての攻撃技を有効化 |
| **弱攻撃** | **M1 / クリック** | 画面タップ / R2 | 素早い低ダメージの連撃 |
| **強攻撃** | **M2 / クリック** | 画面長押し / Y | ガード崩しとフィニッシャー |
| **回避 / ダッシュ** | **Q** | スワイプ / B | 飛んでくるコンボを避ける |
| **ブロック / パリィ** | **F** | 盾アイコン / L2 | ダメージ軽減、パリィの狙い |
| **スプリント** | **Wを2回押し** | ジョイスティック / L3 | 高速移動と追跡 |
| **とどめ / 掴み** | **V** | コンテキストボタン / L1+X | 倒れた相手への追い打ち |

<Callout type="info" title="プロのコツ：シフトロック">
Robloxの設定でシフトロック（Shift Lock）を有効にすることは、ハイレベルなPvPにおいて必須です。これにより、常に相手と正面から向き合うことができ、M1コンボを当てたりブロックのタイミングを合わせたりするのが格段に容易になります。
</Callout>

モバイルプレイヤーの方は、設定メニューでUIの透明度を調整することをお勧めします。これにより、相手の攻撃の予備動作がボタンで隠れて見えなくなるのを防ぐことができます。コントローラーユーザーは、最も反応の良い戦闘の流れを作るためにL2/R2トリガーに集中してください。

</Section>

<Section icon={Target} title="PvP戦闘ガイドと戦略">

**学乱 西片高校**での戦闘は、忍耐と誘い出しのゲームです。むやみな攻撃（いわゆる「M1連打」）は、パリィの仕組みを理解しているプレイヤーに対しては、すぐさま敗北に繋がります。オープンワールドでの学校の喧嘩で勝率を上げるために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="早めに戦闘スタンスに入る">
    紛争地帯の可能性がある場所へ行く際は、必ず **R** を押してから入りましょう。戦闘スタンスは移動速度がわずかに低下しますが、ブロックとダッシュの準備が整い、「初撃」によるスタンロックを防ぐことができます。
  </Step>
  <Step num="2" title="ダッシュダンスを極める">
    **Q** を使って、真後ろではなく横方向にダッシュしましょう。このゲームのほとんどの攻撃は前方への推進力があります。横にダッシュすることで、相手が空振りのモーション中に反撃できる距離を保つことができます。
  </Step>
  <Step num="3" title="パリィを確定させる">
    相手の攻撃の視覚的または聴覚的な合図を待ってから **F** を押します。パリィに成功すると、敵は約1.5秒間無防備になります。これは、フルコンボの弱攻撃や、強力な強攻撃のフィニッシャーを叩き込むのに十分な時間です。
  </Step>
  <Step num="4" title="リセットと再配置">
    コンボを当てた後、すぐに追いかけ回してはいけません。一度下がり、スタミナを回復させ、次の隙をうかがいましょう。絶え間ない攻撃はスタミナ切れを招き、自分自身が動けない標的になってしまいます。
  </Step>
</Steps>

<Callout type="tip" title="高度な戦術：壁ハメ">
学校の狭い廊下では、相手を壁やロッカーに追い込むように位置取りましょう。これにより相手は横ダッシュができなくなり、ブロック（強攻撃で崩せる）するか、コンボを全段受けるかの二択を迫られます。
</Callout>

効果的な戦闘には、引き際を知ることも含まれます。バスケットボールコートや体育館で数的不利に陥った場合は、階段や出入り口などの環境を利用して視線を切り、逃げましょう。サンドボックス型の格闘ゲームにおいて、戦術的な撤退は恥ではありません。

</Section>
